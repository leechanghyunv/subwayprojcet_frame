subway_project_230208는 지하철 하차 어플리케이션입니다. 지하철 정보 관련 어플리케이션이지만 흔한 지하철 노선도가 없습니다. UI를 지하철 노선도 대신 비행기 티켓과 유사하게 제작되어 원하는 목적지 관련 정보의 가시성을 강조했습니다. 이 어플리케이션은 지하철 하차 알림,지하철 시간표,실시간 상하행선 정보,SMS민원 기능이 포함되어 있습니다.
